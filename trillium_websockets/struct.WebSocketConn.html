Struct trillium_websockets::WebSocketConn

source · 
pub struct WebSocketConn { /* private fields */ }
Expand description

A struct that represents an specific websocket connection.

This can be thought of as a combination of a [async_tungstenite::WebSocketStream] and a [trillium::Conn], as it contains a combination of their fields and associated functions.

The WebSocketConn implements Stream<Item=Result<Message, Error>>, and can be polled with StreamExt::next
